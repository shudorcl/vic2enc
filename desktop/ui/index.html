维多利亚2 本地化 CSV / 脚本：游戏格式 (GBK 伪 Latin-1) ⇄ 可读 UTF-8
批量（文件夹）模式：.csv 全部转换；.txt 默认只转 events / decisions 内的脚本，可整体关闭或扩到全部。

    

      提示：游戏读取 cp1252，中文以 GBK 字节伪装成 Latin-1 存储，需配合自定义字体显示。

      解码后得到可读 UTF-8 便于编辑/翻译；编辑完成后再编码回游戏格式覆盖原文件。
    

  

  


  

  

    

      
      
      
    

    

      
      
    

    

      把「PA 汉化」式 mod 迁移到现代 assets/localisation/zh-CN（UTF-8）布局：

      ① 把 localisation/ 内的游戏格式文件解码为可读 UTF-8，镜像写入 assets/localisation/zh-CN/。

      ② 扫描 events / decisions / common 里形如 key = "中文" # English 的脚本行：
      把中文抽出成 English;中文;x 写进同名 .csv，并把脚本行还原为 key = "English"（保留缩进/换行）。

      ③ 改动就地进行，勾选备份会先把 localisation/events/decisions/common 复制到 _pa_backup_*。
